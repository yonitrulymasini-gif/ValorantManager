# 🎮 Valorant Manager

## 📌 Description

**Valorant Manager** est une application permettant de simuler des matchs entre équipes inspirées de l’univers de Valorant.
Le projet offre un système complet de gestion des joueurs avec des fonctionnalités CRUD ainsi qu’un moteur de simulation intégrant des compétences et des événements aléatoires comme les blessures.

---

## 🚀 Fonctionnalités

### 👥 Gestion des joueurs (CRUD)

* ➕ Créer un joueur
* 📄 Lire / afficher les informations
* ✏️ Modifier les statistiques
* ❌ Supprimer un joueur

Chaque joueur possède plusieurs attributs :

* ⚡ Vitesse
* 💪 Force
* 🫀 Endurance
* 🎯 Technique

---

### ⚔️ Simulation de matchs

* Création d’équipes personnalisées
* Simulation automatique des matchs
* Résultats basés sur les statistiques des joueurs
* Système de performance influencé par les compétences

---

### 🩺 Système de blessures aléatoires

* Les joueurs peuvent se blesser pendant les matchs
* Impact sur leurs performances
* Ajoute une dimension stratégique et imprévisible

---

## 🛠️ Technologies utilisées

* 💻 C#
* 🗄️ MySQL
* ⚙️ .NET
* 🧰 Visual Studio Code

---

## ⚙️ Installation

1. Cloner le projet :

```bash
git clone https://github.com/ton-username/valorant-manager.git
```

2. Configurer la base de données MySQL

* Créer une base de données
* Mettre à jour le connection string dans le projet

3. Lancer l’application :

```bash
dotnet run
```

---

## 🔐 Configuration

Exemple de connection string :

```csharp

```
